## Running Tests

Clone Repository

```sh
$ git clone git@github.com:mananrangkuti/sparta.git
$ cd sparta
```

Install dependencies:

```sh
$ npm install
```

Setup Environment

```sh
copy value from .env.sample to .env
```

To run all API test suites, run:

```sh
$ npm run test
```
